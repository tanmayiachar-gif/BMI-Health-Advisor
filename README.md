# BMI Health Advisor

## Overview
A BMI-based health advisory system developed as part of a collaborative academic project to analyze user health data and provide personalized health recommendations.

## Features
- BMI calculation based on user inputs
- Health status classification
- Personalized health recommendations
- Interactive Streamlit-based interface

## Technologies Used
- Python
- Streamlit
- Pandas

## Team Contribution
Worked as part of a collaborative team project involving application development, testing, and workflow implementation.

## Future Improvements
- Integration with fitness tracking systems
- Advanced health analytics
- Real-time data monitoring
